HTML 知识点（三）

Jul 23, 2017

表格标签

表单

表单标题、分区、提交后台地址、提交方式等属性

表单多选框、单选框、上传类型等属性

表单提交、重置属性

表单下拉选框、多行文本框等属性

标签类型

  • 行内元素有：a b span img input select strong
  • 块级元素有：div ul ol li dl dt dd h1 h2 h3 p

html 5 中的属性

  • input type
    • email
    • url
    • number
    • tel
    • search
    • range
    • color
    • date picker(date,month,week,time,datetime,datetime-local)

HTML 语义化

语义化：用正确的标签描述页面

语义化的作用

  • SEO (Search Engine Optimization)
  • 可访问性（照顾残障人士）
  • 可读性（方便团队合作，提高开发效率）

实体字符

实体字符：类似其它语言中的转义字符，用来显示一些 html 可能会忽视的字符。

实体字符表示

  • &entity_name;
    • &nbsp
  • &#entity_number;
    • &#160

常用实体字符

  • 空格 ( &nbsp；)
  • “ ( &quot；)
  • > ( &gt；)
  • < ( &lt；)
  • © ( &copy；)
  • & ( &amp；)

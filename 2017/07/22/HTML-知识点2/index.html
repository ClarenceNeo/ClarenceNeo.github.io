HTML 知识点（二）

Jul 22, 2017

文档章节标签

  • body（页面容器标签）
  • header（页面头部）
  • nav（导航栏）
  • aside（侧边栏）
  • article（独立可重复内容）
  • section（文档中的一节）
  • footer（页面尾部）
  • hx（标题标签）

文本标签

a 标签

创建指向另一个文档的链接

创建一个文档内部的锚点

链接到 email 地址等

文本强调标签

  • em 语义上的强调
  • strong 重要性的强调

    strong > em

行内容器标签

  • span 标签

换行标签

  • br 标签

引用标签

  • cite 引用出处（如作品名称，人物）
  • q 作品中的一段文字（如某人说的一段话）

代码标签

  • code 标签

格式化标签（非强调性质）

  • b 粗体
  • i 斜体

组合内容标签

分区标签 div

  • 用于功能区分区，作为其它标签的容器

段落

  • p 标签

列表标签

  • 无序列表 ul
  • 有序列表 ol

    可以通过type属性控制样式，如：
    <ol type="a" start="2">就是从 b 开始排列

  • 自定义列表 dl
  • pre（已经格式化的内容）
  • blockquote （大段的引用）可以设置 cite 属性，引用的出处

嵌入资源标签

img

alt：描述图片含义，在图片加载失败后显示

iframe 嵌入页面

object,embed 嵌入外部资源

  • objece

    另一种写法（IE8 以下不支持）：

  • embed

video

audio

其他图片标签

  • canvas(基于像素，适用于性能要求比较高的，场景比较复杂的，比如实时数据的展示、游戏等。)
  • svg（矢量，适用于高保真）

热点区域

  • map
  • area

CSS 垂直居中

Jun 26, 2017

“44 年前我们就把人类送上月球了， 但现在我们仍然无法在 CSS 中 实现垂直居中。”

—— James Anderson

在 CSS 中对元素进行水平居中是非常简单的：

  • 如果它是一个行内元素， 就对它的父元素应用 text-align: center ；

  • 如果它是一个块级元素，就对 它自身应用 margin: auto 。

但是一旦涉及到垂直居中绝对是一个难题。

表格布局法

table-cell + vertical-align: middle;

元素宽高未知

基于视口单位的解决方案

使用基于视口高度的单位（vw, vh）。

适用局限，只适用于相对于窗口居中。
